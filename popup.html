What is Apu Extension?

APU Extension is a browser extension that is used on TradingView. APU helps to collect all the backtesting strategy results with preset parameter values.

Guidance to use APU Extension
  1. Make sure you open the strategy on your TradingView.
  2. Open APU Extension.
  3. Go to Strategy Backtest and click "Get Parameter". An excel file containing parameters will be saved.
  4. Adjust the excel file. Leave only the parameters needed for testing.
  5. Click "Insert Parameter" to upload the formulated excel file of parameters.
  6. Click "Test Strategy" to start collecting the results.
What should I be aware of when using APU Extension?
  • Please ensure the browser is not closed when the extension is running.
  • Please always keep a good network connection.
  • Please ensure TradingView is opended in English version.
